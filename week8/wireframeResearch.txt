Pencil: A free desktop application for OS X. Easy to use drag and drop interface, has 100s of different elements including 2d and 3d shapes, various buttons, and text tools. Pinpoint location and sizing tools. Also supports multiple pages. Overall Pencil is a feature rich easy to use drag and drop wire framing tool that I highly recommend. Gliffy: A web based application with a drag and drop interface. Includes many elements such as layout elements called “Swimlanes” and many from elements. Gliffy is an excellent free web based application for wire framing.
